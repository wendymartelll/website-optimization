Project Summary

This is the web optimization project. The goal is to render the web pages as quickly as possible (under one second) using the techniques we learned in the course.

Some of these techniques include:

  • Minifying files
  • Compressing images
  • Inlining and prioritizing critical resources
  • Effectively utilizing HTTP caching
  • Optimizing the Critical Rendering Path (CRP) including:
    • Minimizing overall bytes
    • Reducing the numer of critical resources
    • Shortening the CRP length
